Estamos desarrollando una aplicación web llamada "ControlPyme".

Stack tecnológico:
- Backend: Python con Flask
- Base de datos: SQLite
- Frontend: HTML + TailwindCSS + JavaScript
- Arquitectura: MVC simple

Objetivo:
Sistema de control financiero para múltiples rubros:
- Ferretería
- Apicultura
- Agricultura

Funcionalidad principal:
Registrar ingresos y gastos por rubro y visualizar reportes.

Requisitos clave:
- Código limpio y modular
- Separación clara entre backend y frontend
- Uso de plantillas Jinja2
- API interna con JSON cuando sea necesario
- Interfaz moderna con TailwindCSS
- Preparado para escalar

Estructura de carpetas:

/app
  /static
    /css
    /js
  /templates
  /routes
  /models
  /services
  __init__.py
run.py
